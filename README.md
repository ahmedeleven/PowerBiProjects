## MittWatt

MittWatt is a full-stack web application built using the T3 stack that empowers users to forecast electricity prices in Finland. Developed as part of the LearnIT program at BearIT OY, this collaborative project leverages data-driven insights to help users optimize energy consumption and potentially reduce costs. [Here](https://github.com/ahmedeleven/mittwatt)
